TurtleBot3 Frontier Maze Navigation

Developed and tested a ROS2 TurtleBot3 Burger simulation for autonomous navigation in an unknown maze environment.
• Used frontier-based exploration to identify unexplored areas and navigates automatically.
• Visualised the robot’s map, pose, planned path, and exploration progress in RViz.
• Simulated in a maze environment
• Observed updates on occupancy grid map while robot navigates through different maze sections.

ROS2 TurtleBot3 Frontier Exploration Autonomous Navigation RViz Gazebo

Bluetooth Proximity Strength via Two Raspberry Pis

Built a Bluetooth RSSI-based proximity sensing system using two Raspberry Pis to estimate phone movement and distance.
• Set up two Raspberry Pis to perform synchronised Bluetooth scanning
• Used RSSI readings to estimate phone was stationary, moving closer, or away from Raspberry Pis.
• Implemented discovery and tracking modes to track nearby Bluetooth devices before choosing selected target
• Applied filtered RSSI readings, distance estimation, and confidence indicators to make tracking easier
• Logged Bluetooth readings for post-analysis

Raspberry Pi Bluetooth RSSI Sensing Python Distance Estimation Signal Filtering
